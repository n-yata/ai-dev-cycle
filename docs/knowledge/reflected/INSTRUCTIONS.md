# 反映済みナレッジ（reflected）

`/reflect-knowledge` コマンドによって `docs/rules/` に反映済みのナレッジファイルを格納するディレクトリです。

---

## 目的

- 反映済みナレッジを走査対象から除外し、コマンドの実行速度を維持する
- 反映済みのナレッジも参照可能な状態で保持する

## ディレクトリ構成

```
reflected/
├── INSTRUCTIONS.md
├── design-decisions/     # 反映済みの設計判断ナレッジ
├── review-findings/      # 反映済みのレビュー指摘ナレッジ
├── test-patterns/        # 反映済みのテストパターンナレッジ
└── lessons-learned/      # 反映済みの教訓ナレッジ
```

## 注意事項

- このディレクトリ内のファイルは **参照用** です。削除しないでください。
- 内容の更新が必要な場合は、元のカテゴリディレクトリに戻してから編集し、再度 `/reflect-knowledge` を実行してください。
