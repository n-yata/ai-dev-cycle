## 1. 概要

> システム全体で使用するエラーコードの定義。
> API レスポンスの `error.code` フィールドにこの一覧のコードを使用する。

---

## 2. エラーコード命名規則

```
{カテゴリ}_{詳細}
例: AUTH_TOKEN_EXPIRED, VALIDATION_REQUIRED_FIELD, RESOURCE_NOT_FOUND
```

| カテゴリ | 用途 |
|---------|------|
| AUTH | 認証・認可関連 |
| VALIDATION | 入力バリデーション関連 |
| RESOURCE | リソース操作関連 |
| EXTERNAL | 外部連携関連 |
| SYSTEM | システム内部エラー |

---

## 3. エラーコード一覧

### 認証・認可 (AUTH)

| エラーコード | HTTP ステータス | メッセージ | 発生条件 | クライアント対応 |
|------------|---------------|----------|---------|---------------|
| AUTH_TOKEN_MISSING | 401 | 認証トークンがありません | Authorization ヘッダー未設定 | ログイン画面へ遷移 |
| AUTH_TOKEN_EXPIRED | 401 | 認証トークンの有効期限切れ | JWT 期限超過 | トークンリフレッシュ |
| AUTH_TOKEN_INVALID | 401 | 認証トークンが不正です | JWT 署名検証失敗 | ログイン画面へ遷移 |
| AUTH_FORBIDDEN | 403 | この操作を行う権限がありません | ロール/権限不足 | エラー表示 |

### バリデーション (VALIDATION)

| エラーコード | HTTP ステータス | メッセージ | 発生条件 | クライアント対応 |
|------------|---------------|----------|---------|---------------|
| VALIDATION_REQUIRED_FIELD | 400 |  | 必須フィールド未入力 | フィールドエラー表示 |
| VALIDATION_INVALID_FORMAT | 400 |  | 形式不正（メール等） | フィールドエラー表示 |
| VALIDATION_OUT_OF_RANGE | 400 |  | 値の範囲外 | フィールドエラー表示 |

### リソース (RESOURCE)

| エラーコード | HTTP ステータス | メッセージ | 発生条件 | クライアント対応 |
|------------|---------------|----------|---------|---------------|
| RESOURCE_NOT_FOUND | 404 |  | 指定リソース未存在 | エラー表示/一覧へ戻る |
| RESOURCE_ALREADY_EXISTS | 409 |  | 重複作成 | エラー表示 |
| RESOURCE_CONFLICT | 409 |  | 楽観的ロック競合 | 再取得を促す |

### 外部連携 (EXTERNAL)

| エラーコード | HTTP ステータス | メッセージ | 発生条件 | クライアント対応 |
|------------|---------------|----------|---------|---------------|
| EXTERNAL_SERVICE_ERROR | 502 |  | 外部サービス障害 | リトライ表示 |
| EXTERNAL_TIMEOUT | 504 |  | 外部サービスタイムアウト | リトライ表示 |

### システム (SYSTEM)

| エラーコード | HTTP ステータス | メッセージ | 発生条件 | クライアント対応 |
|------------|---------------|----------|---------|---------------|
| SYSTEM_INTERNAL_ERROR | 500 | サーバーエラーが発生しました | 予期しないエラー | エラー画面表示 |
| SYSTEM_RATE_LIMITED | 429 | リクエストが多すぎます | レートリミット超過 | 時間をおいてリトライ |

## 4. エラーレスポンス形式

### 基本形式

```json
{
  "success": false,
  "error": {
    "code": "VALIDATION_REQUIRED_FIELD",
    "message": "名前は必須です"
  }
}
```

### バリデーションエラー（複数フィールド）

```json
{
  "success": false,
  "error": {
    "code": "VALIDATION_REQUIRED_FIELD",
    "message": "入力値が不正です",
    "details": [
      { "field": "name", "message": "名前は必須です" },
      { "field": "email", "message": "メールアドレスの形式が不正です" }
    ]
  }
}
```
