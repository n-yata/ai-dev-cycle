## 3. 画面遷移図

> Mermaid 記法やテキストベースの図で遷移を表現する。

```mermaid

    ログイン --> ダッシュボード: 認証成功
    ログイン --> ログイン: 認証失敗
    ダッシュボード --> 詳細画面: 項目選択
    詳細画面 --> 編集画面: 編集ボタン
    編集画面 --> 詳細画面: 保存/キャンセル
    ダッシュボード --> [*]: ログアウト
```
